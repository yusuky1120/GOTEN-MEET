# GOTEN MEET

手描きのシェアハウス1階間取りをもとにした、2D俯瞰型フロントエンドプロトタイプです。

## 現在できること

- 仮の1階マップを表示
- アバターをWASDまたは矢印キーで移動
- 外壁・階段・家具との当たり判定
- 現在いる部屋名を画面右上に表示
- 画面サイズに応じた縮小表示

現時点ではフロントエンドのみです。LiveKit、複数人同期、テキストチャット、認証、DBは未実装です。

## ローカル起動

Node.js 20以降を推奨します。

```bash
npm install
npm run dev
```

表示されたローカルURLをブラウザで開いてください。

## 本番ビルド

```bash
npm run build
npm run preview
```
